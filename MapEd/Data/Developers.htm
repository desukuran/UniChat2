UniChat 2

Internet Animation Chatting

Prototype for a Next Generation Internet Community Building System

UniChat was developed by theses members when they were in Samsung SDS. The first version of UniChat was released in Jan. 1997 for the UNITEL, computer online service in Korea. This second version was developed on the concept of quarterview (virtual 3D space) graphics and released in Apr. 1998 for UNITEL. The map editor for the 3D background was also developed for internal use only.

All programs were developed by Microsoft Visual C++ with MFC. 3D graphics were rendered on an SGI machine using Alias.
